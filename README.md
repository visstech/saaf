# SAAF
## Simple AI Agent Framework

An open-source AI Agent Engineering framework.

Learn. Build. Understand AI Agents.

## Vision

To understand and build AI Agents by implementing core components:

- Tool System
- Reasoning Engine
- Planning
- Agent Loop
- Memory Architecture
- LLM Integration

## Current Status

Version: 0.1.0

Completed:

- BaseTool
- ToolManager
- CalculatorTool
- WeatherTool
- EmailTool
- Reasoning Engine Design
- Conversation Memory
- Short-Term Memory

## Architecture

SAAF follows clean software architecture principles:

- Single Responsibility Principle
- Separation of Concerns
- Modular Design
- Storage Independence
